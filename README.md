# BOOKMARKING (Frontend)

책을 읽고 그때 그때 들었던 생각을 바로 바로 남길 수 있는 웹앱

## 기술스택

- React
- Redux
- Typescript
- Storybook

- [x] 책 검색 (KAKAO api)

- [ ] 검색 결과 페이지네이션

- [x] 로컬 로그인
- [ ] 로컬 회원가입
- [ ] 구글 로그인
- [ ] 카카오 로그인

- [x] 리포트 생성

- [ ] 리포트 삭제

- [ ] 리포트 페이지네이션
- [ ] 리포트 디테일
